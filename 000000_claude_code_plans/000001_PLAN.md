# Plan: Ollama + Open WebUI Docker Compose Setup

## Context
Create a bash script and docker-compose file to manage two Docker containers — ollama (LLM server) and open-webui (chat UI) — with simple start/stop/status commands. Running on Linux with Docker Engine. All Docker volumes stored under `/mnt/data/docker`.

## Implementation

### 1. `docker-compose.yml`
- [x] **ollama service**: `ollama/ollama` image, NVIDIA GPU passthrough (`deploy.resources.reservations.devices`), volume bind-mounted from host
- [x] **open-webui service**: `openwebui/open-webui`, maps host port to container port 8080 (open-webui listens on 8080 internally), `WEBUI_AUTH=false`, `OLLAMA_BASE_URL=http://ollama:11434` (service-name resolution via default compose network), volume bind-mounted from host, `depends_on: ollama`
- [x] Uses bind mounts (not named volumes) to `/mnt/data/docker/ollama` and `/mnt/data/docker/open-webui` for explicit host path control
- [x] No `network_mode: host` needed — both services share the default compose bridge network and communicate via service names
- [x] All image names, tags, container names, ports, and volume paths referenced via `${VARIABLE}` interpolation from environment

### 2. `run_ollama_open_webui.sh`
- [x] Variables hoisted at the top of the script:
  - `OLLAMA_IMAGE="ollama/ollama"`
  - `OLLAMA_IMAGE_TAG="0.20.0"`
  - `OPENWEBUI_IMAGE="openwebui/open-webui"`
  - `OPENWEBUI_IMAGE_TAG="0.8"`
  - `OLLAMA_CONTAINER_NAME="ollama"`
  - `OPENWEBUI_CONTAINER_NAME="open-webui"`
  - `OLLAMA_PORT="11434"`
  - `OPENWEBUI_PORT="8080"`
  - `OLLAMA_VOLUME="/mnt/data/docker/ollama"`
  - `OPENWEBUI_VOLUME="/mnt/data/docker/open-webui"`
- [x] Variables exported so `docker-compose.yml` can reference them
- [x] `start` command: runs `docker compose up -d`
- [x] `stop` command: runs `docker compose down`
- [x] `status` command: runs `docker compose ps`
- [x] Usage message printed for no/invalid arguments
- [x] Script resolves `docker-compose.yml` relative to its own location via `-f`
- [x] Script is executable (`chmod +x`)
